Rating Poster Database is a service that adds posters with ratings (IMDB, Rotten Tomatoes, Metacritic, etc) to Jellyfin libraries, it supports many rating types, textless posters, posters translated in many languages, adding badges (4k, HDR, DoVi, etc) and more.

The RPDB plugin requires an API Key to work, you can get one from ratingposterdb.com

Setup Instructions

Only 3 ratings can show at one time on a poster, if one of the first 3 ratings is missing, a backup rating will be used (if available)

Badges will be added to the poster (based on the selected preferences) up to a maximum of 4 badges per poster (depending on what the video supports)
